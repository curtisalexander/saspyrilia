<TabItem value="sas">

```sas
proc import datafile="/some/path/to/person.csv"
            out=ds
            dbms=csv
            replace ;
run;
```

</TabItem>
<TabItem value="py">

```py
# standard library
import csv

with open("/some/path/to/person.csv") as csvfile:
    reader = csv.reader(csvfile)
    for i, row in enumerate(reader):
        print(f"row {i}: {row}")

# pandas
import pandas as pd

df = pd.read_csv("/some/path/to/person.csv")
```

</TabItem>
<TabItem value="r">

```r
library(readr)

df <- read_csv("/some/path/to/person.csv")
```
