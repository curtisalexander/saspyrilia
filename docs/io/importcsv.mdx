Simple examples of importing a csv file.

These are simplistic examples of importing csv files.  Better examples include methods to fix types when importing to ensure consistent dataframes (or returned objects).
:::

<TabItem value="sas">

```sas
proc import datafile="/some/path/to/a.csv"
            out=mycsv
            dbms=csv
            replace ;
run;
```

</TabItem>
<TabItem value="py">

```py
# standard library
import csv

with open("/some/path/to/a.csv") as csvfile:
    reader = csv.reader(csvfile)
    for i, row in enumerate(reader):
        print(f"row {i}: {row}")

# pandas
import pandas as pd

df = pd.read_csv("/some/path/to/a.csv")
```
